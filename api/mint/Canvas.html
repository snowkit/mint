Canvas

mint.Canvas

A canvas is a root object in mint. It requires a rendering instance, and handles all incoming events, propagating them to the children. It also maintains the state of modality, focused controls and dragged contorls to simplify interaction. Additional Signals: none

meta: @:directlyUsed, @:allow(mint.render.Renderer)
